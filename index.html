Sobre Mí

Hola, soy Jesus Tolentino. Un Ingeniero de Sistemas apasionado por desarrollar soluciones tecnológicas innovadoras y eficientes. Tengo experiencia en múltiples lenguajes y herramientas como Java, SQL, MySQL, HTML, CSS, JavaScript y el ecosistema Spring (Spring Framework, Spring Boot y Spring Security). Trabajo con Bootstrap 5 y frameworks visuales modernos, utilizando entornos de desarrollo como NetBeans, Visual Studio e IntelliJ IDEA. Me especializo en crear aplicaciones robustas, escalables y orientadas a resultados.

También tengo experiencia en el uso de herramientas de desarrollo modernas como Git y GitHub para el control de versiones, Docker para la creación y gestión de entornos de desarrollo portables, y Postman para pruebas de APIs REST. Estas herramientas me permiten mantener flujos de trabajo organizados, colaborativos y eficientes.

Mi enfoque se centra en ofrecer productos de alta calidad que no solo cumplan con los requisitos técnicos, sino que también brinden una experiencia de usuario fluida, intuitiva y visualmente atractiva. He participado en diversos proyectos que abarcan tanto el desarrollo back-end como front-end, creando interfaces funcionales y amigables, siempre manteniendo un equilibrio entre el rendimiento, la usabilidad y el diseño.

Creo firmemente que la tecnología avanza a un ritmo acelerado, por lo que me comprometo a mantenerme en constante aprendizaje y actualización. Estar al tanto de las últimas tendencias y buenas prácticas me permite desarrollar soluciones modernas, eficientes y alineadas con las necesidades del entorno tecnológico actual.
